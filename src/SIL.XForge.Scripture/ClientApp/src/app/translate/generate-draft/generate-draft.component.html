Generate back translation drafts

Scripture Forge can assist with drafting a back translation, based on your existing back translation progress. For example, if the translation team has already drafted one or more books and then back translated one or more of them, Scripture Forge can use that example data to draft a back translation of the next book you draft.

Keep in mind that the translation draft will contain mistakes, and you will need to edit the draft that is generated.

The back translation draft will be generated in the background. You can continue to work on other tasks while the draft is being generated. You can also cancel the generation process at any time. The process will typically take at least 8 hours under ideal conditions, but may take longer if the server is busy.
